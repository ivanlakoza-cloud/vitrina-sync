
Патч для vitrina (v2)
---------------------
Что входит:
- app/api/catalog/route.ts — новый формат выдачи API (v2), отключено кэширование Edge/CDN.
- app/page.tsx — новая главная: 6 плиток в ряд, строки под фото как обсуждали.

Быстрая проверка:
1) Откройте https://vitran.ru/api/catalog?v=2 — в ответе должны быть поля api_version: "v2" и build_time.
2) Обновите главную (hard refresh): Ctrl+F5 / Shift+Reload.
3) В шапке справа увидите "build 2025-08-31 07:41:10". Если сменилось — задеплоилось именно это.
